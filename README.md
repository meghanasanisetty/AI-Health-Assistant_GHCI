# AI-Health-Assistant_GHCI

An AI-powered multilingual healthcare assistant designed for rural and underserved communities.
It delivers private, reliable, and easy-to-understand* medical guidance without requiring high digital literacy.

# Problem statement
Existing digital healthcare solutions often depend on constant  English interfaces, and complex apps—factors that exclude millions in rural areas. There is a pressing need for an AI-powered healthcare solution that supports multiple Indian languages and dialects, protects user privacy, and provides simple, actionable guidance to improve community health outcomes.

# Project goal
To make primary healthcare knowledge accessible for everyone by combining AI + local language support.

# Key Features
 Symptom-based intelligent health guidance
 > Multilingual support (Indian regional languages & dialects)
 > Voice + text input for low-literacy users
 > Secure storage — no external data leaks
 > Emergency alerts & guidance when symptoms are critical

# This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

# System Architecture
User Input → Local AI Processing → Symptom Evaluation → Health Recommendation → Online Sync with Supabase.

# How to run locally 
# Clone the repo
git clone https://github.com/meghanasanisetty/AI-Health-Assistant_GHCI.git 
cd AI-HEALTH-ASSISTANT_GHCI

# Install dependencies
npm install

# Start the development server
npm run dev
